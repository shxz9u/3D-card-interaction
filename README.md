웹페이지에 나타나있는 이미지를 바꾸고 싶으시면 원하시는 사진을 디렉토리로 가져와서 card.webp로 바꾸거나 54번째 줄 "card.webp"를 사진 이름으로 변경하시면 되고 사이즈가 안맞다면 스타일 태그의 container과 card의 width, height의 px를 각각 알맞게 수정해주시면 됩니다.<br>
기본사진은 토스유스카드 흰색 입니다.
